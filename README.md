# 😊 CodeAlpha - Sentiment Analysis

## 📌 Description

This project performs sentiment analysis on customer reviews using Natural Language Processing (NLP).

The application classifies reviews as **Positive**, **Negative**, or **Neutral** using the TextBlob sentiment analysis library.

This project was developed as part of the **CodeAlpha Data Analytics Internship**.

---

## 🚀 Features

- Analyze customer reviews
- Positive, Negative & Neutral classification
- Sentiment polarity score
- Beginner-friendly implementation
- CSV dataset support

---

## 🛠 Technologies Used

- Python
- Pandas
- TextBlob
